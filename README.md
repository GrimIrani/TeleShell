# TeleShell
Reverse Shell with aiogram telegram bot.
## Getting started
***Before you go***
Just put your token in API_TOKEN that's in **main.py**
> API_TOKEN = ' ' #<<< Set your bot's API token Here >>>
## Select your way(python or docker)
### (1/2)docker:
```
docker build -t teleshell .
```
### (2/2)python:
```
pip install aiogram
python3 main.py
```
