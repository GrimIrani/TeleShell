## Getting started
***Before you go***
Just put your token in API_TOKEN that's in **main.py**
https://github.com/GrimIrani/TeleShell/blob/b3023e00ec4114e3ceaf21ceb429fda6e05296a5/main.py#L6
#### (1/2)docker:
```
docker build -t teleshell .
```
#### (2/2)python:
```
pip install aiogram
python3 main.py
```
